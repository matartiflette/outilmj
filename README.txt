Instructions :
1. Ouvrez Qt Creator.
2. Créez un nouveau projet Qt Widgets Application.
3. Remplacez les fichiers main.cpp, mainwindow.h et mainwindow.cpp par ceux-ci.
4. Ouvrez mainwindow.ui avec Qt Designer et ajoutez :
   - Un QTabWidget avec trois onglets :
     - Onglet 1 : QPlainTextEdit nommé textStats
     - Onglet 2 : QPlainTextEdit nommé textStuff
     - Onglet 3 : QPlainTextEdit nommé textHistoire
5. Compilez et lancez.
